# Puja Path – Online Pooja Booking Platform

Puja Path is a modern frontend web application that enables users to explore, schedule, and book pooja and religious rituals through a digital platform. The project aims to simplify access to traditional services by providing a clean, intuitive, and responsive interface for devotees.

---

## Features

- Online booking for various pooja and religious rituals  
- Simple, intuitive, and user-friendly interface  
- Fully responsive design for mobile, tablet, and desktop  
- Modern UI built with reusable components  
- Optimized performance using React best practices  

---

## Technology Stack

### Frontend
- React  
- JavaScript (ES6+)  
- Tailwind CSS   
---

## Architecture Overview

### Frontend (React + Vite)

- Component-based architecture for scalability and maintainability  
- Reusable UI components for consistency  
- Responsive design using Tailwind CSS  
- Clean project structure for easier future expansion  

---

## Prerequisites

Before running the project locally, ensure you have the following installed:

- Node.js (v18 or higher recommended)  
- npm
